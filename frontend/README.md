# Dr. Jiii Chatbot Frontend

A ChatGPT-inspired frontend interface with a clean blue and white aesthetic.

## Features

- Clean, minimalist design inspired by ChatGPT
- Responsive layout
- Header with Dr Jii logo and Login/Signup buttons
- Main content area with input field
- Footer with terms and privacy links

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- **Node.js** >= 18.0.0
- **JavaScript** (ES2020+)
- **React** 18
- **Vite** 5
- **CSS3**

## Prerequisites

- Node.js 18 or higher
- npm 9 or higher (or yarn/pnpm)

## Node Version

This project uses Node.js 18. If you're using `nvm`, you can run:

```bash
nvm use
```

This will automatically switch to the correct Node.js version specified in `.nvmrc`.
